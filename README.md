# منظم الدروس - تطبيق إدارة مراكز ودروس المدرسين

تطبيق Flutter يعمل بالكامل **Offline** باستخدام SQLite، لإدارة الطلاب والمجموعات
والصفوف الدراسية والحضور والاختبارات والمدفوعات والحجوزات والتقارير، مصمم خصيصاً
للمدرسين في مصر.

## المزايا الرئيسية

- عمل كامل بدون إنترنت (Offline-first) بقاعدة بيانات SQLite محلية.
- إدارة الصفوف الدراسية والمجموعات مع مواعيد ورسوم إضافية مخصصة.
- إدارة طلاب كاملة: كود فريد تلقائي، ترتيب متعدد، نقل بين المجموعات، أرشفة.
- تسجيل حضور وغياب يومي مع تقارير لكل طالب.
- اختبارات مع رصد درجات وإرفاق صور أوراق الامتحان (تصوير أو من المعرض).
- مدفوعات: اشتراك شهري، رسوم إضافية، خصومات، مع حساب المتبقي تلقائياً.
- تقارير مالية يومية/أسبوعية/شهرية/سنوية وقائمة بالمتأخرين في السداد.
- حجوزات الفصل الدراسي مع متابعة حالة كل حجز.
- بحث شامل وسريع في كل أقسام التطبيق.
- بطاقة طالب بـ QR Code قابلة للطباعة والمشاركة كـ PDF.
- تصدير/استيراد JSON (نسخة احتياطية كاملة) وتصدير CSV لفتحه في Excel.
- دعم كامل للغة العربية و RTL، الوضع الداكن، وتخصيص لون التطبيق وحجم الخط.
- حماية بكود PIN وبصمة الإصبع.

## التقنيات

- Flutter + Material 3
- SQLite (sqflite) + Clean Architecture (Core / Data / Domain / Presentation)
- flutter_bloc (Cubit) لإدارة الحالة
- pdf / printing لإنشاء وتصدير ومشاركة ملفات PDF
- qr_flutter لبطاقات الطلاب

## البناء عبر GitHub Actions

يحتوي المشروع على ملف `.github/workflows/build.yml` يقوم تلقائياً ببناء ملف APK
عند كل Push إلى الفرع الرئيسي، ويمكن تشغيله يدوياً من تبويب Actions
(workflow_dispatch). الناتج (APK) يظهر كـ Artifact قابل للتحميل مباشرة.

## أيقونة التطبيق

الأيقونة موجودة في `assets/icon/icon.png` وتُولَّد تلقائياً لكل أحجام Android
عبر حزمة `flutter_launcher_icons` أثناء عملية البناء.
